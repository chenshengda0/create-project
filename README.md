# create-project

## git
```
    设置git库地址：
        git remote set-url origin git@github.com:chenshengda0/create-project.git
    查看git 配置：
        git config -l
        
    创建并切换分支：
        git checkout -b create-full-app
    查看分支:
        git branch
```
